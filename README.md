# keishin-kofu-site

恵信甲府病院（山梨）の旧サイト情報をもとに作成した、レスポンシブな1ページ構成のリデザインです。

## 構成

- ヒーロー（病院名・150床・主要診療領域）
- 病院概要（運営理念、地域連携）
- 診療・サービス（脳神経外科、リハビリテーション等）
- 患者様の権利と方針
- アクセス・連絡先（病院本体 + 恵信甲府ケアセンター）
- 旧サイトスクリーンショットの参照カード

## ローカルでの確認

```bash
# このディレクトリで
python3 -m http.server 8000
# ブラウザで http://localhost:8000
```
