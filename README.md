# keishin-kofu-site

恵信甲府病院サイトの**シングルページ・リデザイン**です。  
既存情報を整理し、スマホ/PCで読みやすい1ページ構成に再設計しています。

## ローカルでの確認方法

このリポジトリを開き、`index.html` をブラウザで直接開いてください。

- 例: Finder から `index.html` をダブルクリック
- 追加のビルドやサーバー起動は不要

## 追加した主なセクション

- ヒーロー（病院名・概要の導入）
- 病院概要（理念・地域連携の説明）
- 診療・サービス（主要診療領域と提供内容）
- 患者様の権利と方針
- アクセス・連絡先（病院・関連施設情報）
- フッター
